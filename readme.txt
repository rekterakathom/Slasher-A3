The Slasher Arma 3 project is open source. This means anyone can fork and commit to the mission in GitHub https://github.com/rekterakathom/Slasher-A3.
You are allowed to fork and modify this mission, as long as the original authors are credited! (ThomasAngel for the mission, ALIAS for ambient effects)
There are no dependencies, except for 3den enhanced which is an editing dependency, if the mission is edited without it, certain aspects of the mission will be affected.
Mainly the static hunters.
